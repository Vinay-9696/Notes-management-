# 📝 Notes Management System

A simple and user-friendly Notes Management System built using HTML, CSS, and JavaScript. This application allows users to create, edit, delete, and manage notes efficiently with a clean and responsive interface.

---

## 🚀 Features

- ➕ Add new notes
- ✏️ Edit existing notes
- 🗑️ Delete notes
- 💾 Data stored in Local Storage
- 📱 Responsive design
- 🎨 Simple and attractive UI
- ⚡ Fast and lightweight

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 💻 Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/Notes-Management.git
```

2. Open the project folder

```bash
cd Notes-Management
```

3. Open `index.html` in your browser.

---

## 🎯 How to Use

1. Open the application.
2. Enter a note title and content.
3. Click the **Add Note** button.
4. Edit or delete notes whenever needed.
5. All notes are automatically saved in Local Storage.
